## ğŸ’» Getting Started <a name="getting-started"></a>

### Prerequisites

- Code editor: VS code or other
- Web browser: like Google Chrome
- git-scm
```sh
  npm install
```

### Setup

Clone this repository to your desired folder:
```sh
  cd my-folder
  git clone git@github.com:mulugithub/math-magicians.git
```

### Install

To run the project, open the project using a live server from VS code editor.
```sh
  cd math-magicians
  code .
  npm install
```

### Usage
To run this project:
```sh
  npm  start
```
### Run tests
To run tests, run the following command:
To check for html errors run:
```sh
  npx hint .
```
To check for css errors run:
```sh
  npx stylelint "**/*.{css,scss}"
```
To check for js  and jsx errors run:
```sh
  npx eslint "**/*.{js,jsx}"
```
